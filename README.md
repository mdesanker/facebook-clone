# Fakebook

A Facebook clone made with React.

Live demo [here](https://mdesanker.github.io/facebook-clone)

## How to setup

```
git clone https://github.com/mdesanker/facebook-clone.git
cd facebook-clone
npm install
npm start
```

## Built with

- React
- React Router
- styled-components
- Redux
- Firebase - Firestore
